# BramVR Homebrew Tap

Homebrew tap for BramVR CLI tools.

## Install

```bash
brew install BramVR/tap/gohealthcli
```

## Packages

- `gohealthcli` - Local-first, read-only Google Health archive CLI

## Update / Uninstall

```bash
brew update
brew upgrade BramVR/tap/gohealthcli

brew uninstall gohealthcli
```
